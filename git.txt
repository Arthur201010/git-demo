1. 版本號
git version  

2. 設定全域使用者(雲端)
git config --global user.name arthur
git config --global user.email arthur201010@gmail.com

3.初始化倉庫
git init

4.倉庫狀態
git status

git config --list
5.加入控管(暫存區)
git add filename
git add . (所有變動一次加入)

6.四種狀態
U -> Untrack
U -> A (Added)
A -> M (Modified)
A -> D (Deleted)

7.刪除後恢復
git restore filename

8.檢視暫存區
git ls-files -s

9.檢視 object  #7/16 1- 2:05:00
git cat-file -t (型態) shal(前六碼)
             -p (內容) shal(前六碼)

10.加入倉庫儲存
git commit -m "第一次檔案儲存"
            (m:指message)

11.檢視倉庫狀態 # 1- 2:30:00
git log
git log --oneline
git log --graph

12.暫存區檔案恢復(至工作目錄U)
git restore --staged filename

13.倉庫區檔案恢復(至工作目錄U)
git rm --cached filename

14.強制刪除(remove)
git rm -f 4.txt

14-1. git rm 1.txt

15.產生分支 
git branch branch-name

16.檢視分支
git branch

17.切換分支
git checkout branch-name

18.捨棄變更
git checkout . 

19.合併分支(基本上切換master去合併其他分支)
git checkout master
git merge test

20.合併問題?! 1:56:00
同時間修改檔案並commit,合併後會產生衝突,
需進行選擇與確認
git add .
git commit -m "完成合併衝突確認"

21.刪除分支
git branch -D test

22.VIM編輯器用法
1.使用 git commit 進入
2.使用 i 進入編輯模式
3.書寫完畢後使用esc鍵,切換到下方命令列
4.輸入:wq 離開
5.如果寫入失敗輸入:wq!(! =>強制模式)

增加3個檔案跟commit log

24.切換commit-log  #7/18 2:30:00
git checkout 4a8cc8
git checkout master <=切回

25.重置到某一個commit-log
git reset 4a8cc8 
git reset --hard 81c70fe

26.檢視所有歷程
git reflog
